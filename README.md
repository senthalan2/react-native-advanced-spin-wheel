# react-native-advanced-spin-wheel

A production-ready, highly customizable, physics-based spin wheel (fortune/lucky wheel) component for React Native and Expo. Powered by `react-native-reanimated` v2+ and `react-native-svg` for blazing fast 60fps performance.
